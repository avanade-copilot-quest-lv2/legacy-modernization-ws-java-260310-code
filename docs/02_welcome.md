# 02. ウェルカム画面

**対応JSP**: `src/main/webapp/WEB-INF/jsp/bookstore/welcome.jsp`  
**URL**: `/` (ログイン直後の遷移先)  
**必要権限**: ログイン済み  
**Actionクラス**: `com.example.bookstore.action.LoginAction`

---

---

## 表示項目

| # | 項目名 | ソース | 備考 |
|---|--------|--------|------|
| 1 | ユーザー名 | `session["user"]` | |
| 2 | ロール説明 | role → roleDescription 変換 | ADMIN=Administrator, MANAGER=Store Manager, CLERK=Sales Clerk |
| 3 | ログイン日時 | `session["loginTime"]` | yyyy/MM/dd HH:mm 形式 |

---

## ボタン・リンク

| リンク名 | 遷移先 | 条件 |
|---------|--------|------|
| ダッシュボードへ | `/home.do` | 全ロール |
| 販売入力へ | `/sales/entry.do?method=entry` | CLERK / 全ロール |
| レポートメニューへ | `/reports/menu.do` | MANAGER/ADMIN のみ |

---

## 画面遷移

```
自動リダイレクト:
  ADMIN   >>> /admin/home.do     (22. 管理者ダッシュボード)
  MANAGER >>> /reports/menu.do   (17. レポートメニュー)
  CLERK   >>> /sales/entry.do    (07. 販売入力)
  その他  >>> /home.do           (05. ダッシュボード)
```
