# Docs Overview

このディレクトリは、legacy modernization workflow の設計資料を種類ごとに整理したものです。

## 構成

- `docs/state/`
  - 現在の判断状況、次アクション、機械可読な状態
- `docs/decisions/`
  - beta 範囲や運用方針など、合意済みの主要決定
- `docs/spec-drafts/`
  - 重要仕様のレビュー用ドラフト。確定後に必要なら正式な仕様へ昇格する
- `docs/exploration/`
  - リサーチと探索メモ。途中経過も含むが、重要な決定は state / decisions に反映する
