## 書籍検索パネル - 入力項目

| # | 項目名 | name | 型 | 備考 |
|---|--------|------|-----|------|
| 1 | ISBN | `isbn` | テキスト | |
| 2 | タイトル | `title` | テキスト | |
| 3 | カテゴリ | `catId` | セレクト | カテゴリ一覧 (session["categories"]) |

---

## 書籍リストテーブル列

| 列名 | データ | 備考 |
|------|--------|------|
| ISBN | `book.isbn` | |
| Title | `book.title` | |
| Price | `book.listPrice` | |
| Stock | `book.qtyInStock` | OK(緑) / LOW(橙) / OUT(赤) |
| Qty | `qty_{bookId}` (input) | デフォルト1 / 1〜9999 |
| Add | [+Add] ボタン | AJAX: `addToCart(bookId)` |

---

| Subtotal | 単価×数量 | |

---

## ボタン・アクション

| 名前 | 種別 | アクション |
|------|------|---------|
| 検索 | Submit | `/sales/entry.do?method=entry` (GET) |
| クリア | Reset | フォームクリア |
| +Add | ボタン(AJAX) | `/sales/cart/add.do?method=addToCart` (POST) |
| Qty更新 | AJAX | `/sales/cart/update.do?method=updateCart` (POST) |
| × (削除) | ボタン(AJAX) | `/sales/cart/remove.do?method=removeFromCart` (POST) |
| チェックアウト → | Submit | `/sales/checkout.do` |

---

## 在庫表示カラー

| ステータス | 条件 | 色 |
|----------|------|-----|
| OK | 在庫 > 10 | 緑 |
| LOW (低在庫) | 在庫 1〜10 | 橙・太字 |
| OUT (在庫切れ) | 在庫 = 0 | 赤・太字 |

---

## バリデーション (クライアント側)

- ISBN: 13桁チェック
- 数量: 1〜9999 の範囲
- カート空の場合チェックアウト不可

---

## 画面遷移

```
[検索]          >>> 同画面 (書籍リスト更新)
[+Add] (AJAX)   >>> カート更新後 location.reload()
[チェックアウト] >>> /sales/checkout.do   (08. チェックアウト)
未ログイン       >>> /login.do           (01. ログイン)
```
