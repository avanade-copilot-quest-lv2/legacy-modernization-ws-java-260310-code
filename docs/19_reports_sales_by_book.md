## フィルターフォーム - 入力項目

| # | 項目名 | name | 型 | バリデーション |
|---|--------|------|----|--------------|
| 1 | From (開始日) | `startDate` | テキスト | 書式: `yyyy-MM-dd`、必須 |
| 2 | To (終了日) | `endDate` | テキスト | 書式: `yyyy-MM-dd`、必須 |

---

## 結果テーブル列

| 列名 | データ | 備考 |
|------|--------|------|
| # | 行番号 | |
| ISBN | `row.isbn` | 等幅フォント |
| Title | `row.title` | |
| Category | `row.categoryName` | |
| Orders | `row.orderCount` | 右揃え |
| Revenue | `row.revenue` | `$` 付き・右揃え |

最下行: **TOTAL** (合計行)

データソース: `session["salesByBookData"]`
