## 表示項目

| # | 項目名 | ソース | 備考 |
|---|--------|--------|------|
| 1 | エラーメッセージ | `request["err"]` または `javax.servlet.error.message` | |
| 2 | ステータスコード | `javax.servlet.error.status_code` | |
| 3 | エラー発生URL | `javax.servlet.error.request_uri` | |
| 4 | 発生日時 | サーバー日時 | yyyy-MM-dd HH:mm:ss.SSS z 形式 |
| 5 | スタックトレース | `exception.printStackTrace()` | デバッグ用（本番非推奨） |
| 6 | セッションダンプ | セッション全属性 | デバッグ用 |
