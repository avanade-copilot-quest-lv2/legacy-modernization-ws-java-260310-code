## 入力項目 (検索フォーム)

| # | 項目名 | フォームプロパティ | 型 | 必須 | 備考 |
|---|--------|-----------------|-----|------|------|
| 1 | ISBN | `isbn` | テキスト | — | 13桁 |
| 2 | タイトル | `title` | テキスト | — | 部分一致 |
| 3 | 著者名 | `authorName` | テキスト | — | 部分一致 |
| 4 | カテゴリ | `catId` | セレクト | — | カテゴリ一覧から選択 |

---

## 検索結果テーブル列

| 列名 | データ | 説明 |
|------|--------|------|
| # | 行番号 | 1始まり |
| ISBN | `book.isbn` | |
| Title | `book.title` | |
| Publisher | `book.publisher` | |
| Price | `book.listPrice` | `$` 付き右揃え |
| Status | `book.status` | |
| Stock | `book.qtyInStock` | 在庫数 |

## ボタン・リンク

| 名前 | 種別 | アクション |
|------|------|---------|
| Search | サブミット | 検索実行 (GET) |
| Clear | リセット | 入力クリア |
| Home | リンク | `/home.do` |
