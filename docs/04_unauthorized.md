**対応JSP**: `src/main/webapp/WEB-INF/jsp/bookstore/unauthorized.jsp`

## 表示項目

| # | 項目名 | ソース | 備考 |
|---|--------|--------|------|
| 1 | 必要ロール | `request["requiredRole"]` | デフォルト: ADMIN |
| 2 | 現在のロール | `session["role"]` | |
| 3 | アクセス日時 | サーバー日時 | |
| 4 | ユーザー名 | `session["user"]` | デバッグ情報 |
| 5 | クライアントIP | `X-Forwarded-For` ヘッダー or `remoteAddr` | セキュリティ監査用 |
| 6 | 連続試行回数 | `session["_unauthAttempts"]` | セッションごとにカウント |
