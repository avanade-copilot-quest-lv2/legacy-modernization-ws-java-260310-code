# 26. カテゴリフォーム画面 (管理者 - 追加/編集)

**対応JSP**: `src/main/webapp/WEB-INF/jsp/bookstore/admin/category/form.jsp`  
**URL (追加)**: `/admin/category/add.do`  
**URL (編集)**: `/admin/category/edit.do?id={catId}`  
**必要権限**: ADMIN のみ  
**Actionクラス**: `com.example.bookstore.action.AdminAction`

## フォーム入力項目

| # | 項目名 | name | 型 | バリデーション |
|---|--------|------|----|--------------|
| 1 | Category Name | `categoryName` | テキスト | 必須 |
| 2 | Description | `description` | テキストエリア | 任意 |

---

## ボタン

| ボタン | アクション |
|--------|----------|
| Save | フォーム送信 (追加: `/admin/category/save.do` / 編集: `/admin/category/update.do`) |
| Cancel | `/admin/category/list.do` に戻る |

---

## 画面遷移

```
[Save] (追加成功) >>> /admin/category/list.do  (25. カテゴリ一覧) + 成功メッセージ
[Save] (編集成功) >>> /admin/category/list.do  (25. カテゴリ一覧) + 成功メッセージ
[Save] (バリデ失敗)>>> 同画面 + エラーメッセージ
[Cancel]          >>> /admin/category/list.do  (25. カテゴリ一覧)
権限不足            >>> /unauthorized.do        (04. 権限エラー)
未ログイン           >>> /login.do              (01. ログイン)
```
